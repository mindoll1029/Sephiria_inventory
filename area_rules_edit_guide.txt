Sephiria simulator area rule edit guide

실제 실행 파일: index.html
주로 수정할 위치: index.html 안의 const SLATE_RULES = { ... }

1) area 규칙 기본 형식

slate_example: {
  rotate: true,
  position: null,
  effect: { ignoreArtifactRestrictions: false },
  pattern: [],
  area: [
    { type: "belowAll", mark: "minus" }
  ]
}

2) mark 값

plus   = 강화
minus  = 감소/마이너스
ignore = 특수/제약 무시 표시

3) area type 목록

sameRow      = 석판과 같은 줄 전체
sameColumn   = 석판과 같은 세로줄 전체
topRow       = 최상단 줄 전체
bottomRow    = 최하단 줄 전체
aboveColumn  = 석판 위쪽 같은 세로줄
belowColumn  = 석판 아래쪽 같은 세로줄
leftRow      = 석판 왼쪽 같은 줄
rightRow     = 석판 오른쪽 같은 줄
aboveAll     = 석판보다 위쪽 모든 칸
belowAll     = 석판보다 아래쪽 모든 칸
leftAll      = 석판보다 왼쪽 모든 칸
rightAll     = 석판보다 오른쪽 모든 칸
edge         = 가장자리 전체
inner        = 가장자리를 제외한 안쪽 전체
diagonalUp   = 왼쪽 아래 ↔ 오른쪽 위 대각선
diagonalDown = 왼쪽 위 ↔ 오른쪽 아래 대각선
diagonalBoth = 양쪽 대각선 모두
around       = 석판 주변 8칸
all          = 자기 자신을 제외한 모든 칸

4) 예시

응집처럼 석판 아래쪽 모든 칸에 마이너스:
area: [
  { type: "belowAll", mark: "minus" }
]

기반처럼 같은 줄 전체 강화:
area: [
  { type: "sameRow", mark: "plus" }
]

동시성처럼 같은 세로줄 전체 강화:
area: [
  { type: "sameColumn", mark: "plus" }
]

차양처럼 최상단에서만 발동하고 최하단 줄 전체 강화:
position: "topRow",
area: [
  { type: "bottomRow", mark: "plus" }
]

5) 현재 area 규칙이 들어간 석판

agglutination: belowAll minus
justice: sameColumn plus
transition: sameRow plus + sameColumn minus
boundary: edge plus
courage: around plus
miracle: sameRow plus + sameColumn plus
sheen: diagonalBoth plus
base: sameRow plus
concurrency: sameColumn plus
rebellion: diagonalUp plus
shade: position topRow + bottomRow plus

6) 회전 처리

rotate:true인 석판의 area 규칙은 석판 회전에 따라 방향이 바뀝니다.
예: belowAll은 90도 회전 시 leftAll, 180도 회전 시 aboveAll, 270도 회전 시 rightAll로 바뀝니다.
NO_ROTATE_SLATES에 포함되거나 rotate:false인 석판은 area 방향도 회전하지 않습니다.


[경계 석판 예시]
경계처럼 최상단/최하단만 강화하려면 edge가 아니라 topRow와 bottomRow를 함께 사용합니다.

area: [
  { type: "topRow", mark: "plus" },
  { type: "bottomRow", mark: "plus" }
]

edge는 최상단, 최하단, 왼쪽 끝, 오른쪽 끝을 모두 포함합니다.
