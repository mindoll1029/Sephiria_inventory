석판 모양 데이터 자동 반영 안내

이번 버전에서는 /석판.zip 안의 54개 석판 스크린샷에서 초록칸(+), 빨간칸(-), 청록칸(무시)을 자동 추출해 index.html의 SLATE_RULES에 채워 넣었습니다.

수정 위치:
  index.html
  const SLATE_RULES = { ... };

좌표 기준:
  dr:-1 = 석판 기준 위 1칸
  dr: 0 = 같은 줄
  dr: 1 = 아래 1칸
  dc:-1 = 왼쪽 1칸
  dc: 0 = 같은 칸
  dc: 1 = 오른쪽 1칸

mark:
  plus   = 강화
  minus  = 약화/감소
  ignore = 아티팩트 제약 무시 표시

현재 자동 반영 범위:
  - 초록 강화칸
  - 빨간 감소칸
  - 청록 무시칸
  - 회전 불가 목록
  - 차양: topRow 조건 + bottomRow 강화

주의:
  이미지 안의 긴 설명 문장으로만 존재하는 특수 위치 조건은 자동 OCR이 아니라 수동 확인이 필요할 수 있습니다.
  예: 안쪽에 있을 때만, 제일 윗칸에 있을 때만 같은 문장형 제약.
