석판별 강화 모양 데이터가 필요하다는 메시지의 의미

현재 코드에는 석판을 회전할 수 있는지, 위치 제한을 검사하는 구조, 합성된 석판의 규칙을 합치는 구조가 들어 있습니다.
다만 '작동' 버튼이 실제로 어느 칸을 + 또는 -로 표시하려면 각 석판별 강화 좌표가 필요합니다.

필요한 데이터 예시:
SLATE_RULES = {
  slate_example: {
    rotate: true,
    position: null,
    pattern: [
      { dr:-1, dc:0, mark:'plus' },
      { dr:0, dc:1, mark:'minus' }
    ]
  }
}

dr: 석판 기준 위아래 이동값
dc: 석판 기준 좌우 이동값
mark: plus 또는 minus

현재 ZIP의 석판 이미지에는 강화 모양이 보이지만, HTML/JS는 이미지를 자동 판독하지 않습니다.
이미지에서 좌표를 뽑아 SLATE_RULES에 넣어야 '작동' 버튼이 실제 게임처럼 작동합니다.
